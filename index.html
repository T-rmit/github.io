<p class="has-text-align-center has-huge-font-size"><strong>Unreal Engine 4: Project Idea</strong></p>

<!-- /wp:paragraph --></div></div>

<!-- /wp:cover -->


<!-- wp:media-text {"mediaId":131,"mediaType":"image","mediaWidth":25} -->

<div class="wp-block-media-text alignwide is-stacked-on-mobile" style="grid-template-columns:25% auto;"><figure class="wp-block-media-text__media"><img src="https://s3660241.files.wordpress.com/2020/03/unreal.png?w=213" alt="" class="wp-image-131" /></figure><div class="wp-block-media-text__content"><!-- wp:heading {"level":4} -->

<h4>Project Overview:</h4>

<!-- /wp:heading -->


<!-- wp:paragraph {"placeholder":"Content\u2026"} -->

<p>Development of a game using the Unreal Engine software. The game is a fantasy medieval action role-playing game with a focus on making the game difficult for the player keeping the player on their toes.&nbsp;</p>

<!-- /wp:paragraph --></div></div>

<!-- /wp:media-text -->


<!-- wp:heading {"level":4} -->

<h4>The Motivation Behind This Project:</h4>

<!-- /wp:heading -->


<!-- wp:paragraph -->

<p>The motivation for this project is due to my own personal interest in gaming and wanting to one day make a game of my own. This will also enhance my skills with 'Unreal Engine. This Engine is used by a lot of the top AAA games that have come out in the past and will in the future. Some good examples of AAA titles that have used this engine are, , and . (Wiki). The great thing about this Engine is that it's free until your game is developed and making over 3,000 USD quarterly, which is great for a amateur like myself. The software uses a predominantly C++ format when coding. Unreal, also offers a platform to find and/or buy models, game sounds and more for your project (List of Unreal Engine games, 2020).</p>

<!-- /wp:paragraph --></div></div>

<!-- /wp:media-text -->


<!-- wp:heading {"level":4} -->

<h4>Skills Required For This Project:</h4>

<!-- /wp:heading -->


<!-- wp:paragraph -->

<p>The skills required to program in the most optimal way is knowing C++. However, Unreal Engine does offer a lot of blueprints so that people with little to no prior coding knowledge may still be able to use their software. The blueprints are premade blocks of code that you can add to objects to make interactions. Even with minimal knowledge on coding and using the program, 'Unreal Engine' has been out for a long time so there are plenty of guides and tutorials online to help new users (Denham, n.d.).</p>

<!-- /wp:paragraph -->


<!-- wp:heading {"level":4} -->

<h4>Outcome Of This Project:</h4>

<!-- /wp:heading -->


<!-- wp:paragraph -->

<p>If this project is successful, hopefully a well-polished game will be made from it. Even though the aim is to make a game, there are still other outcomes from this project. You get to learn the 'Unreal Engine' software, and how to use at least the basics of it and some other advanced tools. You may even learn a bit of C++ coding through watching online tutorials and guides on how to make certain things in the game. The development of this game could help make future game development of games faster and easier due to prior knowledge.</p>

<h4>What Would My Ideal Job Be In I.T?</h4>

<p>My ideal job in I.T would be working in Cyber Security, more specifically Cyber Security Analysis.<br>An example of this position can be found on Seek and can be accessed <a rel="noreferrer noopener" href="https://www.seek.com.au/job/41162002?type=standout#searchRequestToken=464b53d3-bac1-41d1-b73f-ec1729fcb0f0" target="_blank">here</a>.</p>

<h4>Description Of This Position:</h4>

<!-- /wp:heading -->


<!-- wp:paragraph -->

<p>A 'Cyber Security Analyst' is required to uphold Cyber Security Standards and make sure they are met throughout the whole company. Defend against and simulate cyber attacks to determine ways to protect the company against them. Work with other I.T departments and make sure that Cyber Security Standards are met. </p>

<h4>What Interests You About This Position?</h4>

<p>Cyber Security has always interested me, as I love problem solving, keeping everything up to the required standard and managing things.</p>

<h4>What Does The Position Require?</h4>

<!-- /wp:heading -->


<!-- wp:paragraph -->

<p>To succeed in getting this position I'll need need to have the following experience, knowledge, and traits:</p>

<!-- /wp:paragraph -->


<!-- wp:list -->

<ul><li>Analytical mindset.</li><li>Ability to work in a team.</li><li>Able to communicate with stakeholders that are both technical and non-technical.</li><li>Knowledge of security standards and required frameworks; ISO 27001, PSPF, ISM, VPDSF.</li><li>Provide general security operations.</li><li>Knowledge on Cloud based software.</li><li>Knowledge on managing vulnerability and SIEM tools.</li><li>Required to have 1 or more certifications; CISSP, CEH, CISSP, CEH, ISO 27001, and CompTia Security+</li><li>5 years previous experience in Information Security Roles.</li></ul>
